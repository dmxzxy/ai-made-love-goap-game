# 建筑物改进说明

## 需要重启游戏才能看到效果！

### 主要改进：

1. **建筑尺寸大幅增加**
   - 从 26-38 像素增加到 45-50 像素
   - 更容易看到和识别

2. **更鲜艳的颜色**
   - 金矿：亮金色
   - 所有建筑颜色更明亮

3. **独特形状**
   - 资源类：圆角矩形（金色边框）
   - 防御类：八边形（蓝灰边框）
   - 军事类：菱形（红色边框）
   - 科研类：六边形（紫色边框）
   - 支援类：圆形（绿色边框）

4. **清晰的标签**
   - [RESOURCE] / [DEFENSE] / [MILITARY] / [RESEARCH] / [SUPPORT]
   - 彩色标签，带黑色背景

5. **实时状态显示**
   - 金矿：+8$/s
   - 医疗站：+8 HP/s
   - 维修站：Repair 12/s
   - 武器库：+35% DMG
   - 护盾：50% Shield

6. **更多建筑数量**
   - 金矿：3个
   - 地堡/瞭望塔/路障：各3个
   - 大部分建筑：2个

## 如何查看：
1. 关闭当前游戏
2. 重新运行游戏
3. 开始战斗
4. 等待建筑建造完成（会显示大尺寸、彩色标签、形状）
